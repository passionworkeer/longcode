# longcode 优化计划

> 分析当前竞品，制定优化策略

---

## 📊 竞品分析

### 现有方案

| 方案 | 类型 | 特点 |
|------|------|------|
| **GPT Engineer** | AI 代码生成 | 输入 prompt → 生成整个项目 |
| **Cursor** | AI 编程助手 | 交互式代码补全、Chat |
| **Copilot** | 代码补全 | 实时代码建议 |
| **Devin** | AI 开发者 | 完全自主的 AI 程序员 |
| **Claude Code Skills** | 官方技能 | 各种垂直能力 |
| **Axiom** | 技能集合 | iOS 开发专用技能 |

### longcode 当前位置

**优势：**
- ✅ 完全自动化（比 GPT Engineer 更自主）
- ✅ 任务级别 Git 管理（独特优势）
- ✅ 四重质量关卡（比大多数严格）
- ✅ 失败跳过机制（不会卡住）

**劣势：**
- ❌ 目前只在 Claude Code 中可用
- ❌ 没有可视化的进度条
- ❌ 任务时间估算不智能
- ❌ 没有学习机制
