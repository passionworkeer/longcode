# longcode - 长时间自主开发代理

> 触发方式：slash command `/longcode`
> 描述：启动长时间自主开发流程，从需求收集到代码部署的全流程自动化

---

## 使用方法

```
/longcode [可选：任务描述]
```

或者直接描述你的需求：
```
用 longcode 创建一个实时聊天功能
```

---

## 功能特性

- 🎯 深度需求理解 - 交互式收集 10+ 个关键问题
- 📋 精细任务拆解 - 将大任务分解为 < 2小时 的小任务
- 🤖 按需子代理团队 - 根据任务类型启动相关代理
- ✅ 严格验收流程 - 编译→测试→安全→审查
- 🚫 零用户交互 - 执行期间完全自主
- 📦 自动 Git 提交 - 提交到 feature 分支

---

## 执行流程

1. **需求收集** - 交互式提问（唯一与用户交互的阶段）
2. **生成计划** - 创建详细任务计划
3. **确认执行** - 显示计划后请求确认
4. **自主执行** - 自动完成所有任务（无需用户参与）
5. **完成报告** - 显示结果和遗留问题
