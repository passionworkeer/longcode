# longcode - 完整介绍

> 🤖 Autonomous Development Agent for Claude Code
>
> 只需描述你的需求，longcode 会自主完成从需求分析到代码提交的全流程。

---

## 一句话说明

**longcode** 是一个 Claude Code 技能，让你能像指挥项目经理一样开发软件——只需说出你想要什么，它会自动完成剩余所有工作。

---

## 核心特点

| 特点 | 说明 |
|------|------|
| 🤖 **全自动化** | 说一次需求，全程无需参与 |
| 📋 **智能拆解** | 自动将大任务拆成 < 2 小时的小任务 |
| ⏱️ **时间估算** | 智能估算每个任务和总时长 |
| 📊 **进度可视化** | 实时 ASCII 进度条 |
| 👁️ **预览确认** | 执行前显示要修改的文件 |
| 💰 **成本估算** | 实时 Token 消耗和费用显示 |
| 🛡️ **质量保证** | 编译 → 测试 → 安全 → 审查，四重关卡 |
| 🔄 **容错机制** | 失败自动重试 3 次，仍失败则跳过 |
| 📦 **原子提交** | 每个任务独立 Git 提交，可追溯可回滚 |
| 🚫 **零交互** | 执行期间完全不问你问题 |
| 🎬 **演示模式** | 干跑展示，不实际修改文件 |
| 📝 **完整报告** | 生成计划、执行日志、完成总结 |

## 使用方法

### 触发方式

```bash
# 方式 1：slash command
/longcode

# 方式 2：直接描述需求
用 longcode 创建一个用户认证系统
用 longcode 实现实时聊天功能
```

### 命令行选项

```bash
/longcode [选项]

  -r, --resume      从上次中断处恢复执行
  -s, --status      显示当前执行状态
  -p, --pause       暂停执行
  -t, --max-tasks   限制执行的任务数量
  -d, --dry-run     演示模式，只显示不执行
  -v, --verbose     显示详细日志
  -q, --quiet       只显示关键信息
      --no-preview  跳过预览直接执行
      --no-cost     隐藏成本估算
```

## 验收标准

### 必选项（全部通过才能提交）

- ✅ 代码能编译/构建通过
- ✅ 相关测试通过
- ✅ 安全审查无 High/Critical 问题
- ✅ 代码审查通过（无 Critical 问题）

### 可选项

- ⏹️ 测试覆盖率 ≥ 80%
- ⏹️ 性能指标达标

---

## 问题处理

执行期间**绝对禁止**向用户提问！

遇到问题时：
1. 尝试自己解决（最多 30 分钟）
2. 重试最多 3 次
3. 仍失败则跳过，记录原因
4. 继续下一个任务

最终报告中会说明所有跳过的任务和原因。
